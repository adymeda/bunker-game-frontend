# Bunker Game (Frontend)
This is a frontend part of Bunker Game project that is made using React. The aim of this project is to recreate Bunker board game using web technologies

# Technologies used
### Already implemented
- **Vite** - build tool & development server
- **React** - UI building
- **SASS** - CSS preprocessor
### To be implemented
- **React Router** - application routing
- **Redux/Zustand** - state management
- **Axios** - backend API communication (users, game lobbies, etc.)
- **Socket.IO** - backend communication (game data)
